# Results

Output destination for `src/evaluate_model.py` and `src/benchmark_models.py`:

| Directory | Contents |
|-----------|----------|
| `plots/` | Confusion matrices, training curves, and other generated figures |
| `metrics/` | Saved accuracy/loss/classification-report metrics |

Both directories are currently empty placeholders — populated the next time
those scripts are run. The result figures already referenced from
`docs/architecture.md` (dataset analysis, training curves, quantization, live
prediction) live under `docs/` instead.
